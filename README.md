# ACGN 知識問答小遊戲

以 C++ 打造的 ACGN（動畫、漫畫、遊戲、輕小說）知識問答程式，從 173 題題庫中隨機抽取 30 題進行作答，作答完畢後自動統計成績並給予評比。

---

## 專案背景

我在學校動漫畫社擔任文書長，負責舉辦社團活動。平時我會自行出題，透過 Kahoot 與社員進行知識問答遊戲。題目範圍涵蓋動畫、遊戲、輕小說、輕文學、漫畫等領域，難易度從極難到極易都有，到高二下社團活動結束為止共累積了 173 題。

在學校資訊專題課學習 C++ 陣列時，我想到是否能將這些題目不透過 Kahoot，而是自己寫一個程式來進行問答遊戲。於是我將所有題目放入自行撰寫的陣列中，搭配迴圈與隨機抽題邏輯，完成了這個問答小遊戲。

---

## 功能特色

- **大題庫隨機抽題**：從 173 題中隨機抽取 30 題，每次遊玩體驗不同
- **不區分大小寫**：輸入選項時不需特別區分英文字母大小寫，直接輸入即可
- **自動統計成績**：作答完畢後程式會統計錯題數，並給予評比分數
- **ACGN 知識評比**：根據答對率評估你對 ACGN 領域的了解程度

---

## 題目範圍

| 領域 | 說明 |
|------|------|
| 動畫 | 各類日本動畫作品相關知識 |
| 漫畫 | 日本漫畫、台灣漫畫等作品知識 |
| 遊戲 | 主機遊戲、手機遊戲、PC 遊戲等 |
| 輕小說／輕文學 | 各類輕小說與輕文學作品知識 |
| Vtuber | 虛擬直播主 |

題目難易度涵蓋極易到極難，適合從入門到資深的 ACGN 愛好者挑戰。

## 使用方式

### 編譯

```bash
g++ -o acgn_quiz main.cpp
```

### 執行

```bash
./acgn_quiz
```

### 遊玩流程

1. 程式啟動後自動從題庫隨機抽取 30 題
2. 每題顯示題目與選項，輸入對應的選項字母作答
3. 全部作答完畢後，程式顯示錯題數與評比分數
